# OverGreen

OverGreen is Minecraft mod for Fabric that improves control of game's view. It uses Sodium's config API. This mod is not intended to be unfair, it allows to tweak small visual options to provide satisfying gameplay.

### Download

You can download this mod directly from [Modrinth](https://modrinth.com/project/overgreen).<br>
Dependencies: Fabric API, Sodium.

### Features

- Modifying gamma
- Disabling fog:
  - Overworld fog
  - Nether fog
  - The End fog
  - Water fog
- Customizable permanent HUD
- Hotkey for hiding coordinates from F3 overlay
- Better tooltip for containers like shulker boxes
- Decreasing UI fire when burning
